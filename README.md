## 技术栈

- **前端框架**: Next.js
- **样式**: Tailwind CSS
- **状态管理**: React Context
- **表单处理**: React Hook Form
- **API 集成**: RESTful APIs

## 开始使用

1. 克隆仓库
2. 安装依赖：
   ```bash
   npm install
   ```
3. 运行开发服务器：
   ```bash
   npm start
   ```
4. 在浏览器中打开 [http://localhost:8000](http://localhost:8000)
